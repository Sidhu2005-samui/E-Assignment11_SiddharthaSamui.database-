# E-Assignment11 Documentation

This repository contains various SQL files related to the E-Assignment11 project. Below is the documentation for each SQL file included in this project:

## SQL Files Documentation

1. **create_tables.sql**: This script is responsible for creating the necessary tables for the database.

2. **insert_data.sql**: This file contains scripts to insert initial data into the created tables.

3. **update_records.sql**: This script is used for updating existing records in the database.

4. **delete_records.sql**: This file handles the deletion of records from the database tables.

5. **queries.sql**: This script contains various queries for retrieving data from the database.

Please refer to individual files for more detailed descriptions of their contents and functionalities.
